## Getting and Cleaning Data
### Course Poject

This repository holds all the important components of the coursera project "Getting and Cleaning Data". This course id part of the Data Science Specialisation track.

### Description

- This file **Readme.md** will give the outline of what the other files are about and how they are connected.
- The file **Codebook.md** will give the synopisis of how the main code works.
- The file **run_analysis.R** is the main file in which code is written. This code gives the solution of the problem defined in the project.
- The file **tidydata.txt** is the final data that is got from various resouces given in the question and cleaned according to the instructions and made as a tidy data.
